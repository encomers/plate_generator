# License Plate Generator 🚗

> Генератор автомобильных номерных знаков РФ с поддержкой последовательной и случайной генерации, проверкой дубликатов и HTTP API.

## 📋 Описание

Проект представляет собой сервис для генерации уникальных автомобильных номеров в формате РФ (`У 123 ТЕ 116 RUS`). Поддерживает:

- ✅ Последовательную генерацию номеров (`/next`)
- ✅ Случайную генерацию номеров (`/random`)
- ✅ Проверку дубликатов через репозиторий
- ✅ Потокобезопасность (mutex)
- ✅ Swagger-документацию
- ✅ Middleware: gzip, security headers, логирование

## 🚀 Быстрый старт

### Требования

- Go 1.21+ (используется `math/rand/v2`)
- Make (опционально)

### Установка зависимостей

```bash
go mod tidy
```

### Генерация Swagger-документации

```bash
# Установите swag, если ещё не установлен
go install github.com/swaggo/swag/cmd/swag@latest

# Сгенерируйте docs
swag init
```

### Запуск сервера

```bash
go run cmd/generator/main.go
```

Сервер будет доступен по адресу: `http://localhost:8080`

## 📡 API Endpoints

| Метод | Путь | Описание | Ответ |
|-------|------|----------|-------|
| `GET` | `/next` | Получить следующий номер в последовательности | `У 123 ТЕ 116 RUS` |
| `GET` | `/random` | Получить случайный уникальный номер | `Х 789 УА 116 RUS` |
| `GET` | `/swagger/index.html` | Swagger UI документация | HTML интерфейс |

### Примеры запросов

```bash
# Получить следующий номер
$ curl http://localhost:8080/next
У 001 АА 777 RUS

# Получить случайный номер
$ curl http://localhost:8080/random
К 542 ВМ 777 RUS

# Открыть Swagger UI
# Перейдите в браузере: http://localhost:8080/swagger/index.html
```

## ⚙️ Конфигурация компонентов

### Генератор букв (`LetterGenerator`)

```go
// Создание с алфавитом из 12 разрешённых букв РФ
vocab := []rune{'А', 'В', 'Е', 'К', 'М', 'Н', 'О', 'Р', 'С', 'Т', 'У', 'Х'}
letterGen, err := lettergenerator.New(vocab)
```

### Генератор чисел (`NumberGenerator`)

```go
// Начинает с 0, максимальное значение задаётся в models.MAX_NUMBER_PART_VALUE
numberGen := numbergenerator.New(nil)
```

### Генератор региона (`StaticRegionGenerator`)

```go
// Фиксированный регион, например 777
region, _ := models.NewRegionPart(777)
regionGen := regiongenerator.New(region)
```

### Репозиторий (`MemoryRepository`)

```go
// In-memory хранилище для тестов и разработки
repo := realizations.New()
```

### Сборка PlateGenerator

```go
plateGen, err := plategenerator.New(numberGen, letterGen, regionGen, repo)
if err != nil {
    log.Fatal(err)
}
```

## 🔐 Безопасность и мониторинг

Сервер включает следующие middleware:

| Middleware | Назначение |
|------------|------------|
| `ginzap` | Логирование запросов в формате RFC3339 |
| `gzip` | Сжатие ответов для экономии трафика |
| `secure` | Заголовки безопасности: CSP, X-Frame-Options, X-Content-Type-Options, XSS Filter |

## ⚠️ Ограничения и заметки

- `MemoryRepository` хранит данные только в памяти — при перезапуске данные теряются. Для продакшена реализуйте PostgreSQL/Redis-репозиторий.
- При достижении лимита комбинаций (`overflow`) генератор вернёт ошибку `no more plates available`.
- Все генераторы потокобезопасны благодаря `sync.Mutex`.
